# A Simple App Launcher, That Launches Multiple Executables from Paths!

Honestly this is just a really simple python project I thought I would make public! I haven't been able to find a simple, user-friendly app that does this, so I made it!

## How To Use:
1. Run The .bat file! It automatically checks if you have python installed!
2. Enjoy!

I wanted to make this as simple as possible for people :D

## How To Add Executables!
1. Find the executable you want to add! (This can be really easily done in windows by hitting your windows key, searching for your app, and hitting "Open File Location"
2. After finding the file, right click on it, and hit "Copy As Path"
3. Launch the app, select "View/Edit App List", select "Add An App"
4. For App Name, type whatever name feels appropriate for the app.
5. For App Path, paste the path you just copied
